## About the Project 🌟
This project is a web-based chatbot application that allows users to interact with an AI model. It provides a clean and intuitive interface for sending messages and receiving responses. The chatbot supports text input and also allows users to attach files (images, PDFs, TXT, CSV) with their queries. It features a theme toggle for both light and dark modes, a history of conversations, and the ability to clear the chat.

    ```bash
    cd chatBOT
    ```
2.  **Obtain a Groq API Key:**
    -   Visit the [Groq API website](https://console.groq.com/keys) and sign up or log in.

3.  **Configure API Key:**
    -   Open the `script.js` file.
    -   Locate the `API_KEY` constant and replace `"secret_key_here"` with your actual Groq API key:
        ```javascript

        ```
4.  **Open in Browser:**
    -   Open the `index.html` file directly in your web browser.

## Usage 💡
This chatbot is designed for general conversation and information retrieval. You can use it for:

-   **Asking Questions:** Get answers on a wide range of topics.
-   **Generating Ideas:** Brainstorm creative ideas or solutions.
-   **Code Assistance:** Ask for explanations or suggestions related to coding.
-   **Drafting Content:** Get help in writing or refining text.

**To interact with the chatbot:**

1.  **Type your message** in the input field at the bottom and press Enter or click the send icon.
2.  **Click on the predefined suggestions** to quickly start a conversation.
3.  **Attach files** (images, PDFs, .txt, .csv) by clicking the attach icon.
4.  **Toggle between light and dark themes** using the sun/moon icon.
5.  **Stop a response** if it's taking too long or you change your mind.
6.  **Clear the entire chat history** if needed.

## How to use 💡
This section elaborates on the practical steps to engage with the chatbot, building upon the general usage instructions.

1.  **Initiating a Conversation:**
    *   Simply type your query into the text box labeled "What can I help you with?" and hit the send button (arrow icon).
    *   Alternatively, click on any of the "suggestion cards" (e.g., "Design a home office setup...") to pre-fill the input and send the message.

2.  **Attaching Files:**
    *   Click the paperclip icon (`attach_file`) next to the input field.
    *   A file input dialog will appear. Select your image, PDF, TXT, or CSV file.
    *   The selected file will be previewed (if it's an image) or its name displayed.
    *   Send your message along with the attached file. The AI will process the context provided by the file along with your text prompt.
    *   To cancel an attached file before sending, click the close icon (`close`).

3.  **Managing Responses:**
    *   During a response generation, a "stop circle" icon (`stop_circle`) will appear. Click this to abort the AI's current response.
    *   Responses are displayed with a typing animation for a more engaging experience.

4.  **Interface Controls:**
    *   **Theme Toggle:** The `light_mode`/`dark_mode` icon allows switching between visual themes.
    *   **Delete Chats:** The trash bin icon (`delete`) clears the entire conversation history.

## API Reference 🔗
This project integrates with the Groq API to power its AI chat capabilities.

-   **API Endpoint:** `https://api.groq.com/openai/v1/chat/completions`
-   **HTTP Method:** `POST`
-   **Authentication:** Bearer Token (`Authorization: Bearer YOUR_GROQ_API_KEY`)
-   **Model Used:** `llama-3.3-70b-versatile`
-   **Key Parameters:**
    *   `messages`: An array of message objects representing the conversation history (`role`: "user" or "assistant", `content`: message text).
    *   `temperature`: Controls the randomness of the output (set to `0.7`).
    *   `max_tokens`: Maximum number of tokens to generate in the response (set to `1024`).

*Note: The API key should be kept secret and is configured directly in `script.js`.*
